# Multifactor Participation

A governance protocol for verifying human judgment in consequential AI operations, modeled on multifactor authentication.

## Contents

- [Paper](paper.md) — Full paper: problem, related work, methexis, protocol specification, verification controls, worked example, design implications, limitations.
- [Appendix: Record Schemas](appendix-schemas.md) — Minimum record structures for participation records, accountability records, spirit-letter rationales, and execution tokens.

## Core Claim

Human-in-the-loop governance verifies presence, not participation. MFP proposes three non-substitutable verification factors — computational, participatory, and accountability — enforced at execution time with auditable artifacts. Some tasks require accountable human judgment by nature, not merely because current models are imperfect. MFP is a protocol for verifying that the judgment was present.
